# Using NttNotes

## Installation
- Install the package on your environment:
see here https://learn.microsoft.com/en-us/dynamics365/fin-ops-core/dev-itpro/deployment/apply-deployable-package-system for detail
## Use
1) Start the Dynamics 365 F&O web client
2) Go To *'Main menu/Modules/System administration/Inquiries/NttNotes'*
3) Choose *'Table documentation'* or *'Data entity name'*
4) Fill up the form:
    - Select the table/view you want to document
    - Check the option *'Show system fields'* if you want to include the system fields in the report
    - Check the option *'Recursive'* if you want to go over the first level of relation (this can take time to complete!)
    - Click *'Ok'* button
5) Wait for the procedure to complete in background
6) Open the report (Excel file) and see the results
